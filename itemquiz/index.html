Guess the Item!

Guess the item based on the description given from the wiki. It is not case sensitive, so you can guess "The Sad Onion", "the Sad onion", "tHe sAd onIon", etc. and still get it right. Note: Due to how this game is coded, all (added in repentance) or things like that have been removed, so the item description may have multiple effects, such as items having "tear delay down" and also "tears up".

Type "more" in the box to get the first letter of the item, at the cost of half a point.
